Test Plan — How to Verify Each Component
1
VIX (Market Volatility)
Expected range: 12–80. Below 20 = calm market. Above 30 = fear.
Verify: curl http://localhost:5000/api/macro → check vix.value
Cross-check: Compare with Yahoo Finance ^VIX
✓ Pass if value is a number, available = true, latency < 2000ms
2
Put/Call Ratio (via yfinance SPY options chain)
Expected range: 0.5–1.5. Below 0.7 = bullish (calls dominate). Above 1.0 = bearish.
Source: SPY nearest-expiry options chain. CBOE CSV was deprecated (returns 404).
Verify: curl http://localhost:5000/api/test-components | python3 -m json.tool
✓ Pass if source = "yfinance/SPY", value between 0.4–2.0
3
IV Skew (Options Fear Gauge)
Ratio: 5%-OTM put implied volatility ÷ ATM call IV.
Normal: 1.0–1.3. High skew (>1.3) = options market pricing in fear.
Today on Pi: python3 -c "from etf_bot import get_iv_skew; print(get_iv_skew())"
✓ Pass if value is between 0.7–2.5, available = true
4
RSI — 14-day Relative Strength Index
Range: 0–100. Below 30 = oversold (bullish reversal). Above 70 = overbought (bearish).
Per-ticker; tested here with SPY as proxy.
Verify: curl http://localhost:5000/api/etf/SPY | python3 -m json.tool | grep rsi
✓ Pass if value between 0–100, latency < 3000ms
5
MACD (12, 26, 9) — Trend Confirmation
Histogram = MACD line minus signal line. Positive = bullish trend.
Key signal: cross = 1 means bullish crossover just happened (strong buy signal).
Tested with SPY as proxy (each ETF gets its own MACD).
Verify: curl http://localhost:5000/api/etf/SPY → signal.factors.macd
✓ Pass if histogram has a numeric value, available = true
6
Yield Curve (10yr − 3mo Treasury Spread)
Positive = healthy economy. Negative (inverted) = recession signal.
Sources: ^TNX (10yr) and ^IRX (3mo) from Yahoo Finance.
Current spread shown in macro strip. Today's value: check the strip or run tests.
✓ Pass if ten_yr and three_mo are both populated, latency < 3000ms
7
Crypto Fear & Greed (0–100)
Source: alternative.me API (free, no key). 0–24 = Extreme Fear, 75–100 = Extreme Greed.
Carries 40% weight for crypto signals.
Verify: curl https://api.alternative.me/fng/?limit=1
✓ Pass if value 0–100, label matches range
8
5-day Price Momentum
Percentage change in price over last 5 trading days, normalised to [-1, +1].
Strong short-term directional signal. Per-ticker; tested here with SPY as proxy.
Verify: curl http://localhost:5000/api/etf/SPY | python3 -m json.tool → factors.momentum
✓ Pass if value between -1 and +1, available = true
9
News Sentiment (TextBlob on RSS feeds)
Fetches Reuters, MarketWatch, WSJ RSS + Reddit (r/investing, r/wallstreetbets).
Polarity: -1 (very negative) to +1 (very positive). Article count shows coverage quality.
Verify: curl http://localhost:5000/api/etf/SPY → check articles array length
✓ Pass if article_count > 3. Warn if 0 (RSS blocked on Pi-hole)
10
Market Indices (DJIA, NASDAQ, S&P 500)
Real-time price + daily % change via yfinance (^DJI, ^IXIC, ^GSPC).
Shown in the indices strip on the Signals tab. Informational only (not in signal formula).
Verify: curl http://localhost:5000/api/market-indices
✓ Pass if all 3 indices return price and change_pct, latency < 3000ms
